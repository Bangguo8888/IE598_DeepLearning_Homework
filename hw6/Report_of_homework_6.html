import re
import matplotlib.pyplot as plt
def get_data(filename):
    with open(filename,'r') as f:
        texts = f.read()
        
    lines = texts.split('\n')
    episodes = []
    scores = []
    for line in lines:
        score = re.search('RScore:\s+(-?\d+.\d+) RPPS',line)
        if score:
            scores.append(float(score.groups(1)[0]))
            episode = re.search('Episode:\s+(\d+) Score:',line)
            episodes.append(int(episode.groups(1)[0]))
    return episodes,scores

And we can also see that, when $\beta = 1$, the Rscores converges very quickly, probably after 1000 episodes and the converged Rscores is pretty low, which is about 130. While when $\beta = 0.00$, it needs about 10000 episodes to become stable and the Rscores are about 900.While when $\beta = 0.01$, the Rscores increases exponentially before 15000 episodes.After that it becomes stable and the Rscores are about 1750.
